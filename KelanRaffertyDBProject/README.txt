INTRODUCTION
------------

The Dental Administration SQL database builds and populates the entire administrative database
(using SQL commands) written in scripts, providing administrators simplistic access to all data.
Other commands may also be used internally to connect the databse at build time.

 * For a full version of the module, visit the project page:
   https://www.drupal.org/project/admin_menu
